# SMSGGDJ HELP screen text -> build/help.inc via tools/makehelp.py (run by make).
# One line per screen row. Rules:
#   - blank line          = a blank row
#   - '---' on its own     = page break (new HELP page)
#   - '@VERSION'           = the live version + build stamp, drawn by the ROM
#   - leading spaces are kept (indent); trailing spaces are trimmed
#   - the 'HELP n/N' title row is drawn by the ROM, not here
#   - UPPERCASE only (the font has no lowercase)
#   - <= 20 chars/line (the Game Gear window width); makehelp errors if longer
# Edit this file and rebuild -- no assembly changes needed to change the text.
