SMSGGDJ sample patcher

1: load your .sms or .gg (128 KB). 2: drop sounds into a kit — any format the browser can decode. 3: trim, drive, audition. 4: download the patched ROM. Nothing leaves this page.

The pool is 8 kits of 8 samples. On a KIT instrument the KIT field picks which kit; the note maps chromatically to the 8 slots — slot 0 = the lowest note, wrapping every octave. Empty slots are silent. Each kit can be saved/loaded on its own as a .smskit file to reuse across ROMs.
